# NewsTLDR Admin


## Install

#### 1. clone this repo

#### 2. Install

` pip install -r requirements.txt `

#### 3. Edit API endpoint in `mambo.yaml`

#### 4. Run the API service endpoint

#### 5. Serve locally

`mambo serve`

#### 6. Build to deploy

`mambo build`

#### 7. Setup S3lify to deploy to S3

`s3lify setup`

#### 7.1 Publish

`s3lify publish`
